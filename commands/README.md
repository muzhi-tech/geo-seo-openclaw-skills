# Commands

快捷命令目录，提供常用 SEO/GEO 操作的快捷方式。

## 计划中的命令

| 命令 | 描述 | 状态 |
|------|------|------|
| `/geo-audit` | GEO 快速审计 | 🚧 计划中 |
| `/seo-check` | SEO 快速检查 | 🚧 计划中 |
| `/schema-gen` | Schema 生成 | 🚧 计划中 |
| `/llms-gen` | llms.txt 生成 | 🚧 计划中 |

## 使用方式

```bash
# GEO 审计
/geo-audit https://your-site.com

# SEO 检查
/seo-check https://your-site.com/page
```
