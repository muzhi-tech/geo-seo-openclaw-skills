# SEO & GEO Skills

本目录包含所有 SEO 和 GEO 相关的技能模块。

## 已有技能

### geo-site-audit

独立站 GEO 全站审计技能，基于 SHEEP 框架：

- **功能**: 全站 GEO 分析、平台优化建议、Schema 生成
- **触发词**: "GEO审计"、"AI可见性分析"、"全站GEO优化"
- **使用**: `/geo-site-audit https://your-site.com`
